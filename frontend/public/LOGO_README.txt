Put your two logo image files here so they always appear in the app header:

1. contractor-logo.png   (or .jpg) - Contractor logo (left side in LTR)
2. owner-logo.png        (or .jpg)   - Owner logo (right side in LTR)

Supported formats: PNG, JPG, SVG (use .png or .jpg in filename).
If a file is missing, that logo is hidden automatically.
